== MiniCard Theme for WordPress ==

DEMO: http://minicard.mikejolley.com/
More info: http://mikejolley.com/projects/minicard/
WordPress.org page: http://wordpress.org/extend/themes/minicard

A business card theme for WordPress with built in vCard/hCard support, inspired by the excellent Business Card site by Tim Van Damme (http://timvandamme.com/). To configure the theme go to admin > theme options after activating the theme.

Created by Mike Jolley (http://mikejolley.com)
